# 截图文件夹

请在此文件夹放置项目截图，用于 README.md 展示。

## 需要的截图

1. **main-interface.png** - 主界面
   - 展示搜索框
   - 展示语言切换按钮
   - 展示整体界面设计

2. **strategies.png** - 推荐策略选择
   - 展示三种推荐策略的选择界面
   - 展示知识图谱关系类型选择

3. **keywords.png** - 关键词选择
   - 展示智能提取的关键词标签
   - 展示用户选择关键词的界面

4. **results.png** - 推荐结果
   - 展示推荐书籍列表
   - 展示推荐理由和匹配度评分

## 截图建议

- 分辨率：1920x1080 或 1280x720
- 格式：PNG（保证清晰度）
- 内容：使用真实数据，展示实际效果
- 语言：可以截取中文和英文两个版本

## 如何截图

1. 启动服务：`python start.py`
2. 访问：http://localhost:5000
3. 使用浏览器的截图工具或按 F12 打开开发者工具截图
4. 保存到此文件夹，命名按上述要求

## 更新 README

截图完成后，README.md 中的图片链接会自动生效：

```markdown
![主界面](screenshots/main-interface.png)
```
